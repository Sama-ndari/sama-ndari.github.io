NotifSpy: Notification Tracker

An Android utility app that silently captures every notification on your phone and lets you recover deleted or dismissed messages. Built with Flutter for the UI and a native Kotlin NotificationListenerService for real-time interception, bridged via platform channels.

How It Works:

  • Native Listener - Kotlin service runs in the background, intercepting every notification posted or removed by Android
  • Platform Bridge - MethodChannel for permission management and EventChannel for real-time streaming of notification data to Flutter
  • Local Storage - All captured notifications stored in Hive, fully offline with no cloud dependency

Key Features:

  • Deleted Message Recovery - Captures notification content before it's removed, so you can read deleted WhatsApp/Telegram messages
  • App Filtering - Browse notifications grouped by source app, with dedicated views for WhatsApp, Telegram, and messaging apps
  • Contact Filtering - Filter within an app by specific contact or conversation to find exactly what you need
  • Search & Stats - Full-text search across all notifications, with counters for total captured and deleted messages
  • Dark-First UI - Material Design 3 theme with purple/cyan accent palette, optimized for AMOLED screens
